EV‑MTD ：运动感知事件‑帧检测与跟踪基准 : Motion-aware Event‑Frame Detection & Tracking

EV‑MTD 是首个 大规模、高分辨率、像素级对齐，并提供 目标级运动状态标签 的事件‑帧目标检测与跟踪基准。 它在多种光照与运动工况下刻画跨模态特性，为鲁棒感知与融合算法提供统一评测平台。

EV‑MTD is the first large-scale, high-resolution, pixel-aligned event‑frame benchmark with per-object motion state labels for object detection and tracking. It characterizes cross-modal behavior under diverse illumination and motion conditions, enabling fair evaluation of robust perception and fusion methods.

数据集 Dataset

在城市道路、高速路、隧道等驾驶场景中采集，覆盖弱光、过曝、高速运动等困难工况。数据以统一结构组织，便于复现与对比。

Collected across driving scenarios (urban, highway, tunnels), covering low light, overexposure, and fast motion. Data is organized in a unified structure for reproducible benchmarking.

对齐与标定 Alignment

EV‑MTD 使用共视场的事件相机与 RGB 相机，并通过标定/立体校正与时间同步实现两种模态的像素级一致性。

EV‑MTD uses an event camera and an RGB camera with shared FoV, achieving pixel-level consistency via calibration/rectification and time synchronization.

  • 一致的检测框投射到 Event / RGB。
  • Consistent box projection to Event/RGB.
  • 为轨迹提供运动状态（轨迹级/帧级可扩展）。
  • Motion states for tracks (track-level / frame-level extensible).

运动状态 Motion States

EV‑MTD 为每条目标轨迹提供离散运动状态标签，将“运动语义”引入事件‑帧检测与跟踪评测，便于研究机动目标与快变运动场景。

EV‑MTD labels each object track with discrete motion states, introducing motion semantics into event‑frame detection and tracking for maneuvering and rapidly changing motion.

基准任务 Benchmark Tasks

  • 多目标跟踪：HOTA / MOTA / IDF1。
  • MOT: HOTA / MOTA / IDF1.
  • 状态识别：Acc / F1。
  • State recognition: Acc / F1.
