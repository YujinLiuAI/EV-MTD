EV‑MTD: 运动感知事件‑帧检测与跟踪基准 A Motion-aware Event‑Frame Detection & Tracking Benchmark

EV‑MTD 是首个 大规模、高分辨率、像素级对齐，并同时提供 运动状态标签 的事件‑帧目标检测与跟踪基准。 它系统性地刻画物体在多种运动模式（静止、匀速、加减速、机动转向等）下的跨模态特性， 为鲁棒视觉感知和跨模态融合算法提供统一评测平台。

EV‑MTD is the first large-scale, high-resolution, pixel-aligned event‑frame benchmark with per-object motion state labels for detection and tracking. It systematically characterizes cross-modal behavior under diverse motion patterns (static, cruising, acceleration, braking, maneuvering), providing a unified benchmark for robust perception and multimodal fusion.

数据集概览 Dataset Overview

EV‑MTD 通过统一的采集平台，在城市道路、高速路、隧道、交叉路口等场景中持续采集事件‑帧数据， 覆盖从弱光、高亮、逆光到复杂运动的多种工况。借鉴 PEOD 的场景多样性设计与 DSEC 的驾驶场景设定，我们进一步引入目标级运动状态标注， 为时空建模提供更细粒度监督。

EV‑MTD is collected on a unified platform across urban roads, highways, tunnels, and intersections, covering low light, overexposure, backlight, and complex motions. Inspired by the scene diversity of PEOD and the driving setup of DSEC, we additionally provide per-object motion state labels for finer-grained supervision of spatiotemporal models.

相机与对齐系统 Camera & Alignment System
Pixel‑aligned Event‑Frame

借鉴 PEOD 的共轴双相机系统与 DSEC 的精确标定流程，EV‑MTD 使用事件相机与 RGB 相机共视场配置， 通过标定与时间同步实现：

Following the coaxial dual-camera design of PEOD and the accurate calibration of DSEC, EV‑MTD uses an event camera and an RGB camera with overlapping FoV and achieves:

  • 像素级空间对齐（经标定与立体校正）。
  • Pixel-level spatial alignment via calibration and rectification.
  • 微秒级时间同步（硬件触发 / 时间戳对齐）。
  • Microsecond-level temporal synchronization.
  • 统一的目标检测标注在 Event & RGB 图像中的一致投影。
  • Consistent projection of detection boxes on both Event & RGB views.

这使得研究人员可以在完全对齐的事件‑帧视角下， 分析不同运动模式下的跨模态表现，并设计鲁棒的融合模型。

This enables analysis of cross-modal behavior under different motion patterns on a fully aligned event‑frame view, and the design of robust fusion models.

目标级运动状态 Per‑Object Motion States

与多数仅提供几何标注的基准不同，EV‑MTD 为每条目标轨迹提供离散运动状态标签，将“运动语义” 引入事件‑帧检测与跟踪任务，有助于研究例如“机动目标”、“急刹车”、“缓加速”等关键行为。

Unlike benchmarks with purely geometric labels, EV‑MTD attaches discrete motion state annotations to each track, introducing "motion semantics" into event‑frame detection and tracking, and enabling research on critical behaviors such as maneuvering, hard braking, and gentle acceleration.

状态定义 State Definition
Per‑track Annotation
State Description
Static 目标基本静止，相对运动接近 0。 Object remains nearly static, negligible relative motion.
Cruise 匀速运动，速度平稳。 Constant velocity, smooth motion.
Accel 加速状态，速度随时间显著增加。 Acceleration, speed increases notably over time.
Brake 减速 / 刹车状态，速度显著下降。 Deceleration or braking, noticeable speed drop.
Maneuver 机动状态，伴随明显转向或复杂轨迹变化。 Maneuvering with significant steering or complex paths.
  • 状态标签在轨迹级别给出，并可投射到每一帧。
  • States are defined per-track and can be propagated to each frame.
  • 可用于多任务学习（检测 + 跟踪 + 状态分类）。
  • Suitable for multi-task learning (detection + tracking + state classification).
  • 适合研究事件数据在复杂运动下的优势场景。
  • Helps explore where event data excels under complex motion.
基准任务 Benchmark Tasks
Detection & Tracking

EV‑MTD 提供以下基准任务与建议指标：

EV‑MTD provides the following benchmark tasks and metrics:

  • 事件‑帧目标检测：在对齐的 Event / RGB 图像上预测检测框。 指标：mAP、mAP@IoU=0.5:0.95。
  • Event‑Frame Object Detection on aligned Event/RGB images. Metrics: mAP, mAP@IoU=0.5:0.95.
  • 多目标跟踪（MOT）：基于事件‑帧输入输出时序轨迹。 指标：HOTA、MOTA、IDF1 等。
  • Multi‑Object Tracking (MOT) with event‑frame inputs. Metrics: HOTA, MOTA, IDF1, etc.
  • 运动状态识别：为每条轨迹/每个时间步预测离散运动状态。 指标：Accuracy、F1‑score。
  • Motion State Recognition for each track/time step. Metrics: Accuracy, F1‑score.
  • 联合任务：检测 + 跟踪 + 状态联合学习，探索多任务协同收益。
  • Joint Tasks: detection + tracking + state classification in a unified model.

我们鼓励研究者设计跨模态时空模型， 深入利用事件流在高动态范围与快运动场景中的优势， 类似于 PEOD 展示事件在极端条件下的鲁棒性。

We encourage researchers to design multimodal spatiotemporal models that exploit the advantages of event streams in HDR and fast motion scenarios, similar to how PEOD demonstrates robustness under extreme conditions.
